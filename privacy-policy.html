개인정보처리방침

최종 수정일: 2026년 4월 7일

Spotify Controller(이하 "본 확장 프로그램")는 사용자의 개인정보를 중요하게 생각합니다. 본 방침은 확장 프로그램이 어떤 데이터를 처리하며, 이를 어떻게 보호하는지 설명합니다.

1. 수집하는 데이터

본 확장 프로그램은 다음 데이터를 처리합니다:

  • Spotify OAuth 토큰 — Spotify 계정 인증을 위해 발급받는 액세스 토큰과 리프레시 토큰
  • 재생 상태 정보 — 현재 재생 중인 트랙, 아티스트, 앨범 정보
  • 즐겨찾기 상태 — 트랙의 라이브러리 저장 여부

2. 데이터 저장 방식

  • 모든 데이터는 사용자의 브라우저 내(chrome.storage.local)에만 저장됩니다.
  • 외부 서버로 데이터를 전송하거나 수집하지 않습니다.
  • Spotify API와의 통신은 Spotify 공식 API 엔드포인트를 통해서만 이루어집니다.

3. 데이터 공유

본 확장 프로그램은 사용자의 데이터를 제3자와 공유하지 않습니다. 수집된 데이터는 확장 프로그램의 기능 제공 목적으로만 사용됩니다.

4. 데이터 삭제

확장 프로그램을 제거하면 브라우저에 저장된 모든 데이터가 자동으로 삭제됩니다. 로그아웃 시 OAuth 토큰이 즉시 삭제됩니다.

5. 필요한 권한

  • identity — Spotify OAuth 인증
  • storage — 토큰 및 재생 상태 로컬 저장
  • alarms — 재생 상태 주기적 폴링
  • host_permissions (api.spotify.com, accounts.spotify.com) — Spotify API 호출

6. 문의

개인정보 처리에 관한 문의사항은 GitHub Issues를 통해 연락해 주시기 바랍니다.
